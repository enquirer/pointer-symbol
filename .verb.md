## Usage

If you have any rendering issues on your platform, please create an [issue or pull request](../../issues).

```js
var pointer = require('{%= name %}');

console.log(pointer());
// cygwin, msys, win32 
// '>'

// linux
// '‣'

// everything else
// '❯'

console.log(pointer({small: true}));
// cygwin, msys, win32 
// '»'

// linux
// '‣'

// everything else
// '›'
```
